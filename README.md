# Product API (Go)

##  Описание

Учебный мини-проект на Go, реализующий простой HTTP API для получения информации о продукте по id.

Проект предназначен для практики:
- HTTP handlers
- работы с query-параметрами
- валидации входных данных
- написания unit-тестов для API

---

##  Эндпоинт

### GET /product

Получение информации о продукте по идентификатору.

#### Query-параметры

| Параметр | Тип | Обязательный | Описание |
|---------|-----|--------------|---------|
| id | int | +            | Идентификатор продукта |

Пример запроса:
```bash
GET /product?id=1
```
---

##  Ответы сервера

###  Успешный ответ — 200 OK

```json
{
  "id": 1,
  "name": "Book",
  "price": 100
}
```

-------------------------------------
### Ошибки

Статус                 |    Когда возвращается
-----------------------|---------------------------------
400 Bad Request        |    id отсутствует
400 Bad Request        |    id не число
400 Bad Request        |    id <= 0
404 Not Found          |    продукт с таким id не найден
405 Method Not Allowed |    используется не GET

----------------------------------------------------
### Бизнес-логика

Функция GetProduct:

• проверяет валидность id

• возвращает продукт для id = 1 и id = 2

• возвращает ошибку, если продукт не найден

-------------------------------

### Тестирование

Реализованы unit-тесты для handler’а с использованием пакета net/http/httptest.

Покрыты сценарии:

• валидный запрос

• отсутствующий id

• id не число

• отрицательный id

• несуществующий продукт

• неверный HTTP-метод

Запуск тестов:
```bash
go test ./..
```
Используемые технологии

• Go

• net/http

• encoding/json

• testing

• httptest
